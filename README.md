# 員工線上打卡紀錄程式

## 功能

- 管理員 / 員工分權
- 管理員可建立「員工名單」與員工打卡密碼
- 員工打卡頁（`/kiosk`）只需輸入：員工編號 + 密碼 + 上下班類型
- 員工打卡頁提供清楚欄位標示與大字即時日期時間
- 打卡成功後會自動顯示：員工姓名 + 打卡成功畫面
- 只有管理員可下載打卡紀錄（Excel `.xls`）
- 管理員每次修改打卡紀錄都會寫入稽核紀錄

## 預設管理員

- 帳號：`ADMIN`
- 密碼：`admin123`

## 使用方式

1. 啟動程式

```bash
python app.py
```

2. 管理員登入後台：`/login`
3. 到「建立員工名單」頁面：`/admin/employees`
4. 建立員工編號、姓名、密碼
5. 員工到打卡頁：`/kiosk`
6. 輸入員工編號 + 密碼即可打卡，系統會顯示員工姓名與成功畫面

## 啟動補充

- 若 `5000` 連接埠已被占用，程式會自動改用下一個可用連接埠（例如 `5001`）。
- 也可自行指定連接埠：`PORT=5050 python app.py`
