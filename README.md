# 影片人臉重複辨識工具

這是一個 Python Web 程式，提供「拖曳影片上傳」介面，並分析影片中的人臉是否重複出現。

## 功能

- 拖曳上傳影片（或點擊選擇檔案）
- 自動偵測人臉
- 分群判斷是否為同一人
- 顯示每位人物的出現次數與是否重複出現

## 安裝需求

建議使用 Python 3.10+

```bash
pip install opencv-python numpy
```

## 啟動方式

```bash
python app.py
```

開啟瀏覽器：`http://127.0.0.1:5000`

## 注意事項

- 目前使用 OpenCV Haar Cascade + 特徵向量相似度做人物分群，屬於輕量版本。
- 影片解析度、角度、光線、遮擋都會影響辨識結果。
